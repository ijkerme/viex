Five command modes:

Insert mode:

Press i enter in INSERT MODE whenever at COMMAND MODE.

It executes no command when it in INSERT MODE, and all of actions executed by netbeans.

 

Command Mode:

Press ESC enter in COMMAND MODE whenever at any a mode.

 

Visual mode:

Press v enter in VISUAL MODE when at command mode.

 

Visual Block:

Press b enter in VISUAL BLOCK MODE when at visual mode.

 

Misc mode:

Press t enter in  MISC MODE when at command mode.

 

 

 

 

 

 

Features Highlights:

Commands in COMMAND MODE:

h     Move cursor a line up

j   Move cursor a line down

h   Move cursor a char left

l   Move cursor a char right

cj           Copy a line down

ck          Copy a line up

mj      Move a line down

mk          Move a line up

 

new        New a file

       Create a new file or open a existed file, or create a file from existed a file template.

Command "new ..\Test.i" creates a new file with a existed file template the preceding dir of at the current dir.

  Suffix i of the file "Test.i" indicates to create a java interface. The following codes map is default in viex:

  c indicates to create a java class

  e indicates to create a java enum

  a indicates to create a java annotation

  p indicates to create a java empty class

  m indicates to create a java class with a main method
